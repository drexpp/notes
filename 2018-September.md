### C++
  
  #### [Bitwise operations](https://youtu.be/q5GcA0m8q_Q?t=9m29s)
  
  ```
  char c1 = 35; // 0001 0011
  char c2 = (1 << 5) | (1 << 1) | (1 << 0); // same 35 value
  char c3 = c2 & ~(1 << 5); //inverse of (1 << 5) operation & used to change 5th bit to zero.
  ```
